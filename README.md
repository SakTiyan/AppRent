# AppRent

Experience Without Buying

## About

AppRent adalah aplikasi sistem sewa iPhone yang memungkinkan pengguna menyewa perangkat iPhone dengan mudah, cepat, dan fleksibel tanpa harus membelinya.

## Features

- Manajemen data iPhone
- Sistem penyewaan
- Riwayat transaksi
- Dashboard admin
- Manajemen pengguna

## Tech Stack

- Laravel
- Bootstrap
- MySQL
- jQuery / JavaScript

## Installation

```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
